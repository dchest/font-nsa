NSA Smiley Font

Because there is no better smiley to use when you need to talk about breaking into Google to spy on people.

Download

How to use in documents or presentations

Download, extract, and then install the TrueType font by double-clicking on font-nsa.ttf and, usually, clicking "Install". Select NSA font in your text editor or PowerPoint and type s to get the smiley .

How to use on the Web

Include font-nsa.css in your HTML between <head> and </head>:

<link rel="stylesheet" href="font-nsa.css" type="text/css">

Insert smiley with the following construction:

<i class="nsa nsa-smiley"></i>

Don't forget to upload fonts directory.
